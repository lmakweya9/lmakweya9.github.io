Mini Task Manager

Modular Desktop Productivity Tool

Python 3.x Persistent Storage Modular Design CLI / GUI

Project Overview

The Mini Task Manager is a desktop-based application designed to demonstrate the fundamentals of Software Engineering. It provides a reliable way to manage daily tasks using a clean interface, ensuring that data is never lost between sessions.

Technical Achievements

  • Persistent Data Handling: Implemented a robust file-handling system to ensure tasks are saved to local storage, allowing the application to maintain state even after a reboot.
  • CRUD Architecture: Designed the application logic to follow the CRUD pattern, providing a seamless user experience for adding, viewing, editing, and removing tasks.
  • Modular Codebase: Organized the project into distinct modules for data storage, business logic, and user interaction, facilitating easier debugging and future feature expansion.
