Freelance Management Dashboard

Secure Full-Stack Financial Hub

React Frontend FastAPI Backend JWT Auth PostgreSQL

Architecture Overview

Developed to replace manual spreadsheet tracking, this application provides a centralized source of truth for freelance earnings and client statuses.

Technical Achievements

  • Security First: Implemented JWT-based stateless authentication and Bcrypt password hashing to ensure data integrity.
  • Live Financial Tracking: Designed a reactive budget engine that aggregates revenue in real-time.
  • Decoupled Deployment: Orchestrated a CI/CD strategy using Vercel (Frontend) and Render (Backend).
