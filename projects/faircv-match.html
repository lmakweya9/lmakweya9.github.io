FairCV-Match

An interpretable Python NLP tool that matches CVs to job descriptions using scikit-learn. Includes performance and fairness visualizations for responsible AI.

Features

  • CV to Job Description matching
  • Fairness-aware machine learning
  • Performance and interpretability visualizations
  • Scikit-learn based ML pipeline

Tech Stack

Python, scikit-learn, Pandas, NLP, Data Analysis
