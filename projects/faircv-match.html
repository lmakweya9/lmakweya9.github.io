FairCV-Match

Interpretable NLP for Recruitment Optimization

Python Scikit-Learn NLP (TF-IDF) Machine Learning

Project Overview

FairCV-Match is a specialized Natural Language Processing (NLP) tool designed to bridge the gap between candidate resumes and job requirements. Unlike "black-box" AI, this tool focuses on interpretable matching to ensure recruiters understand why a candidate was ranked highly.

Technical Achievements

  • Semantic Vectorization: Implemented TF-IDF (Term Frequency-Inverse Document Frequency) to transform raw text into mathematical vectors, prioritizing unique skills over common stop-words.
  • Cosine Similarity: Developed a scoring engine using Cosine Similarity to measure the distance between CV vectors and Job Description vectors, providing a percentage-based match.
