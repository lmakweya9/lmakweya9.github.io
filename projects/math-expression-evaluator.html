Math Expression Evaluator

Decoupled Logic & Scientific Computing Engine

Python 3.x OOP Architecture Unit Testing (Pytest) Tkinter UI

Project Overview

This project is a deep-dive into software engineering principles. I engineered a professional-grade scientific calculator by strictly decoupling the mathematical engine from the user interface. This ensures the evaluation logic can be reused in web or mobile apps without modification.

Technical Achievements

  • Variable State Management: Developed a custom parser that handles variables (e.g., storing x = 15) and constants, allowing for multi-step scientific calculations.
  • CI/CD Pipeline: Integrated a GitHub Actions workflow that runs a comprehensive suite of unit tests on every push to ensure mathematical accuracy.
